Pada kesempatan kali ini, kami akan membahas sebuah data relasional. Kami akan menggunakan paket nycflights13 di RStudio. Dan data informasi penerbangan akan di publikasikan pada aplikasi ini dengan tujuan untuk mengetahui informasi penerbangan dari bandara di Kota New York.  

Dataset ini menggunakan sebuah kombinasi dataset dari library (nycflights13) dengan Data penerbangan terbuka ( https://openflights.org/data.html )
