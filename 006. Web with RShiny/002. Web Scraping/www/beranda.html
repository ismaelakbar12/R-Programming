Dev : Ismail Akbar | Lecturer : Prof. Suhartono, S.Si, M.Kom

Pada kesempatan kali ini, kami akan membahas sebuah teknik web scraping. Web scraping adalah proses mengekstrak data dari sebuah situs web. Salah satu contoh web scraping adalah menyalin daftar kontak dari direktori web. Kami akan menggunakan paket rvest di RStudio untuk pengambilan data secara online. Dan data informasi Jaringan Keselamatan Penerbangan akan di publikasikan pada aplikasi ini dengan tujuan untuk mengetahui informasi kejadian kecelakaan penerbangan dari seluruh kejadian.  

Dataset ini diambil dari sebuah situs ( http://aviation-safety.net )
